# single-vue-page
single-vue-page, webpack project template for vue

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run start
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

## Description

### 目录结构

build 单独的构建配置目录
src 源码目录文件夹
